Admin Page

OVERVIEW: In this lecture we discuss implementing an admin page to enable some users to administer other users. The admin page will allow the admin user create, update, and remove existing users. Admin users can list all other users and manage their information. The admin page is protected using passport JS so that only users that are logged in and are in the admin role can access the page. The page communicates with the Web service using a client service. The server model is refactored to support retrieving list of users for the admin user to have access to all the users.
