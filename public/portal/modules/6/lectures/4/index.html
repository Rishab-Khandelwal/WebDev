Update User

OVERVIEW: This lectures discusses handling post requests on the server, parsing the request body for user information, inserting a new user into the database, and navigating programmatically using the $location service.

The server is configured with the body-parser module to parse the request body into a JSON object. Mongoose is used to establish a database connection, and create a data model to interact with the database. Mongoose provides a low level API for interacting directly with the database. A data model provides a high level API to the rest of the application.
