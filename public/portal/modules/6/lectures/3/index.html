Navigation

OVERVIEW: This lecture covers communication between the client service and Web service. The client encapsulates all server communication in a singleton client service. The controller uses the service to send new user information for adding a new user in the server. The client service posts the request to the server along with the new user information embedded in the body of the request. The Web service parses the request body for the new user information. The new user is then added to the database.
