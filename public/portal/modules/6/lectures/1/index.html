Register

OVERVIEW: These videos document implementing user registration. A registration page gathers new user information and is sent to a user Web service. The Web service uses a user model to check to see if the user already exists. If the user doesn't already exist, the Web service uses the model to create a new user. After the user is created the client navigates to the profile.
